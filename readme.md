# Cross-Modal Architectural Retrieval

## Downloading the dataset

The Floor PlanCAD dataset can be downloaded from the following link: [Floor PlanCAD Dataset](https://floorplancad.github.io/). This dataset contains a variety of floor plans in SVG format, which will be used for training and evaluating our cross-modal retrieval system.

Download training and testing datasets from the above link and place them in a directory named `data/` within your project structure. Training data should be placed in `data/train/` and testing data in `data/test/`.
